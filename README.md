# RabbitVpnDemo
a demo project for testing iOS network extension with NEkit
- 一个使用NEKit的iOS App Demo

这个分支用于测试VPN连接状态，不需要引入NEKIT库 同时也不能科学上网，仅用于测试vpn建立
